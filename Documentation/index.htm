Installation

Unzip the file that you download after purchase. The following files and folders will be in the unzipped file:

  • flatmarket.zip (Packed theme file for install)
  • flatmarket folder - theme for manual FTP upload
  • Dummy Data Import folder - Demo data package for quick start
  • Documentation (This file)
  • PATTERNS - background patterns for shop (FREE BONUS)

The theme can be uploaded in two ways:

  • FTP upload: Using your FTP program, upload the non-zippedflatmarketfolder into the /wp-content/themes/ folder on your server.
  • WordPress upload: Navigate to Appearance > Install Themes > Upload. Go to browse, and select the zipped flatmarket.zip file. Hit Install Now and the theme will be uploaded and installed.

Once the theme is uploaded, you need to activate it. Go to Appearance > Themes and activate FlatMarket theme.

If you need help installing Wordpress see the link here : http://codex.wordpress.org/Installing_WordPress

After the install finish, go to FlatMarket Control Panel page, then click Save button import default settings values to database.

PLUGINS INSTALLATION

After you Activate this theme you will see message in WP admin panel that you must install required plugins and can install recommended plugins. Don't worry - all plugins included with theme and you don't need to download something.

  1. Click Begin installing plugins in this message
  2. You must check all required plugins and can check recommended plugins (we recommend it to use, but theme will work without this plugins too):
  3. Select Install in dropdown Bulk Actions menu and click Apply
  4. You should get All installations have been completed message with installed Plugins table below it
  5. Go to Plugins again, and Activate all installed plugins
  6. If you installed recommended WooCommerce plugin you will see this notice:
  7. If you will import our sample data later your should click Skip setup button. If you will configure all site pages manualy click Install WooCommerce Pages button to create basic shop pages.

IMPORTANT: If you have see error that plugin folder for "FlatMarket Visual Page Builder" already exist this mean you already have installed some version of this plugin with some other theme before. In this case:

  1. Go to Plugins section in WP Admin panel.
  2. Find "WPBakery Visual Composer" plugin (it may have another name too, depends of how another theme developer name it, but it will be some visual page builder name).
  3. Click Deactivate near it.
  4. Click Delete near it and confirm plugin deletion.
  5. Now install our FlatMarket Visual Page Builder plugin.

HOMEPAGE SETUP

First of all you need to create required pages and select it for frontpage display:

  1. Go to Pages > Add New
  2. Input some page name, for example Home (we will fill it some steps later)
  3. Scroll down page to see Page Settings box and add homepage to Page css class option.
  4. Check Don't display this page title option, because we don't need to see Homepage title:

  5. Publish homepage page
  6. Go to Pages > Add New
  7. Input some page name for blog, for example Blog and Publish it
  8. Go to Settings > Reading
  9. Choose Front page displays option "A static page", select Home page that you just created for Front page option and Blog page for Posts page option:

  10. Click Save settings

Now you can create content of your HomePage. You can skip this step and read next documentation section "menu management" if you imported sample data, because you already will have demo homepage as on our demo site.

You can add different blocks for homepage with our predefined styles to made your homepage looks similar to our demo site or create homepage as you want in our Visual Page Builder (all content on our demo site created with it).

Lets try to create our sample homepage blocks one by one to understand how its work:

ADDING LATEST PRODUCTS GRID

You can display latest products from WooCommerce on your Home Page. If you already imported our sample dummy data you will have sample products and categories in your shop, if not you need to add several products and categories in WooCommerce.

  1. Add row in Visual Page Builder
  2. Click big + button to add content to this row and select Text Block from Page Builder elements list
  3. Switch text editor to Text mode and paste this WooCommerce shortcode to content area:

    <h2>Latest products</h2>
    [recent_products per_page="15" columns="5" orderby="date" order="desc"]
  4. You can change titles and text here, change per_page limit (this is products limit in grid), sort order.
    Don't change columns count (this will not take effects in this theme).
  5. Save changes
  6. Click Column settings button:

  7. Add products-module to Extra class name:

  8. Save settings
  9. You can Update page and check how your changes look on frontend Home Page now

ADDING FEATURED PRODUCTS GRID

You can display featured products from WooCommerce on your Home Page. If you already imported our sample dummy data you will have sample products and categories in your shop, if not you need to add several products and categories in WooCommerce.

First of all you need to set some of your products as Featured to see it in list:

  1. Go to Products > Products
  2. Find product in list that you want to check as Featured and click star icon near it:

  3. That's all! Now this product set as featured

Now you can continue edit your Homepage:

  1. Add row in Visual Page Builder
  2. Click big + button to add content to this row and select Text Block from Page Builder elements list
  3. Switch text editor to Text mode and paste this WooCommerce shortcode to content area:

    <h2>Featured products</h2>
    [featured_products per_page="15" columns="5" orderby="date" order="desc"]
  4. You can change titles and text here, change per_page limit (this is products limit in grid), sort order.
    Don't change columns count (this will not take effects in this theme).
  5. Save changes
  6. Click Column settings button.
  7. Add products-module to Extra class name.
  8. Save settings
  9. You can Update page and check how your changes look on frontend Home Page now

TWITTER setup (TWITTER PRO WIDGET CONFIGURATION)

This theme have built in Twiiter Pro widget to display your tweets in widgets areas. If you installed it you need to configure it.

New Twitter API 1.1 requires authefication for your application/website, and you need to create new Twitter app in Twitter API for this:

  1. Go to http://dev.twitter.com/ and login with your Twitter user
  2. Go to your applications (https://dev.twitter.com/apps) and click Create a new application button
  3. Fill your website details in form:

  4. You will see your app details page.
  5. Click "Create my access token" button at the bottom of page. After some minutes you can refresh page and see your Twitter App security details that you will need to setup any Twitter plugin.
  6. Go to Settings > Twitter widget
  7. Copy and paste your Consumer key and Consumer secret from your twitter app page:
  8. Click Update options button on bottom.
  9. Click Authorize New Account button. Login with your Twitter account.
  10. Now you can use this widget in Widgets.
